        PREPARANDO O NOSSO AMBIENTE DE DESENVOLVIMENTO

    Para entendermos como funciona a linguagem de programação java é importante aprendermos 
    o seu funcionamento no hard code, sem usar IDE como eclipse 

    Para isso vamos usar o notepad ++, bloco de notas ou o VS code

    1-> Instalar o VSCODE ou Notepad++
    2-> Instalar o java com jre e jdk
    3-> preparar a variaveis de ambiente do windows para aceitar os comandos: java e javac
    vamos usar para executar nosso código no prompt de comandos

                PREPARANDO AS VARIAVEIS DE AMBIENTE DO WINDOWS 
    
    Após a instalação, é necessário configurar as variáveis de ambiente para que o Windows reconheça os comandos java e javac.

    Adicionando ao PATH
    Abrir Configurações do Sistema:

    No menu Iniciar, pesquise por "Variáveis de Ambiente" e clique em "Editar as variáveis de ambiente do sistema".
    
    Editar a variável Path:

    Na seção Variáveis do sistema, encontre a variável chamada Path e clique em Editar.
    Clique em Novo e adicione o caminho para a pasta bin do JDK. Exemplo:
    C:\Program Files\Java\jdk-XX.X.X\bin

    o mesmo para o caminho onde está o jre
