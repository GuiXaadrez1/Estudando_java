                            INTRODUÇÃO!
    
    Java é uma linguagem puramente tipada, perfeita para Programação Orientada a Objeto.
    Programas em java -> são compostos por um ou mais classes.
    Classes -> É a estrutura, módelo, forma de um Objeto! É o conceito
    Objetos -> É a materialização da classe, o que é conceito se torna "real".

                        REGRAS DA LINGUAGEM

    Todos os identificadoes(variáveis, constantes, nome de classes, métodos, packages e etc).
    São sensíveis ao contexto(CASE SENSITIVE), isto é, diferencia letras maiúsculas de minúsculas.

    EXEMPLO: Saldo != saldo e saldo != saldo

    Um par de chaves ({}) delimita o corpo de toda definição de classe, métodos ou instruções em 
    blocos(condicional, repetição, etc...). As intruções (comandos) devem terminar com pontos e vírgulas (;).
